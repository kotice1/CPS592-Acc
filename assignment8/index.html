About this app

This is a calculator application. It can be used by clicking the buttons with the mouse or following the instruction below to use your keyboard.

  • Use the 'w', 'x', 'a', 'd' keys to navigate the calculator buttons. When the desired key is highlighted, use the 'f' key to select that value. This is the 5-key application.
  • The 3-key application works the same way and uses the 'j' and 'l' keys to navigate left and right and the 'h' key to select.
